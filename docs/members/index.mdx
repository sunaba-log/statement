## メンバーを追加する方法

新しいメンバーを追加するには、以下のファイルを編集してください：

1. **`src/data/members.ts`** - メンバー情報を `members` 配列に追加
2. **メンバー画像** - `static/img/members/` フォルダに画像を配置
3. **自動反映** - このページが自動的に更新されます

### 追加例

```typescript
{
  id: "new-member",
  name: "新規 メンバー",
  nameEn: "New Member",
  role: "エンジニア",
  roleEn: "Engineer",
  department: "バックエンド チーム",
  joinDate: "2026-03-01",
  bio: "プロフィール説明...",
  image: "/img/members/new-member.jpg",
  skills: ["Skill1", "Skill2"],
  specialty: ["特技1", "特技2"],
  hobbies: ["趣味1", "趣味2"],
  github: "username",
  twitter: "username",
  website: "https://example.com",
}
```
